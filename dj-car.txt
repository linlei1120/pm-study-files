嘀加智享车生活后台（dj-car-life-system）
dj-car-life-system 是嘀加智享车生活 SAAS+ 企业级 Web 管理后台，面向园区店、服务商、平台运营等角色，覆盖商品、订单、财务清分、驻店人员、销售、预警等全链路运营能力。产品形态是 单页应用（SPA），与微信小程序工程无关。
技术栈
类别	技术
核心框架
Vue 2.6 + Vue Router 3 + Vuex 3
UI
Element UI 2.13 + D2Admin
构建
Vue CLI 5 + Webpack 5
网络
axios + 自研 ROP 网关与多签名方案
运行环境
Node 14.21.3
架构特点
多网关 API：dj.api.smartcarlife.*、risk.manage.api.*、scrm.api.* 等按 method 前缀路由到不同 axios 实例，并做 SHA1/MD5 签名与 Session 注入。
权限驱动路由：登录后从 Vuex 读取权限源，动态生成菜单与可访问页面。
D2Admin 封装：d2-container、d2-table、d2-form 等统一布局与表格/表单规范。
工程化：Jest 单元测试、Playwright E2E、生产环境 Sentry 监控、OSS 上传等。

一句话总结
这是一个基于 Vue 2 + Element UI + D2Admin 的 大型车生活 SaaS 运营后台，通过模块化路由与统一 API 层，支撑从商品上架、订单履约、财务清分到驻店绩效、销售分析、智能预警等 20+ 业务域 的日常运营与配置管理。
